# Definitions and Acronyms

| Term | Definition |
|------|------------|
| GPU | Graphics Processing Unit, used for computational tasks, often associated with cloud credits. |
| LLM | Large Language Model, a type of artificial intelligence model for natural language processing. |
| LLM Coin | A unit representing usage of LLM models within the platform. |
| PRD | Product Requirement Document. |
| RFS | Ready For Service, the target date for a feature to be deployed and available. |
| SRS | Software Requirements Specification. |
| Superadmin | A user role with elevated privileges, capable of managing voucher creation and distribution. |
| Voucher Product | Refers to the broader category of vouchers that can be generated for various offerings within the Cloudeka environment. |
